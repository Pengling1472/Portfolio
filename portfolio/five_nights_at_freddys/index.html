Source Code

All characters, music, and sound effects in this project are owned by Scott Cawthon and Nintendo and the game's concept originates from Scott Cawthon. This project has been developed collaboratively by myself and my team. I programmed the majority of the game using JavaScript, and I've listed some of the features I've implemented below.

Gameplay

  • Collectible coins
  • Door functionality
  • Time and power mechanics
  • Minecraft Server UI for game settings and shops
  • Minecraft Experimental Camera for the game's camera
  • JSON UI for the game's camera and displaying player coins
  • Animatronics jumpscare
  • In-game events

Arcades & Bosses

  • Items functionality for saving progress and teleporting to other players
  • JSON UI for player's health and death screen
  • Boss behavior and attack patterns
