Parkour Tag

Home

The idea of the game is not ours and it's from MCC's Parkour Tag. The project was made by a team of two and below it is shown both of our prespectives on playing the game.

Original Map

We originally made this map, but we did notice that the size of the build is too large for a parkour tag. I used command blocks to run the game and made a compass that the hunter obtained which points near by runner. I also made a spectator mode using command blocks that will be set to the runner if they lost ( this was before spectator mode was added to Minecraft Bedrock ).

Replicated Map

The map was from the Internet and was not made by us. After learning Minecraft Scripting API, I programmed the game with Javascript and used Minecraft Server UI Module to have UI for team selection and game settings, coded the countdown and timer for the game.
